How it works

Three steps, no subscriptions, no data leaving your laptop.

1

Install & run

Clone the repo, run pip install -e . and post-pulse. A local server starts on port 8080 — no cloud account needed.

2

Log into LinkedIn once

Click Sync Now. A real Chrome window opens and asks you to log in. Post Pulse saves your session for 30 days so you won't be prompted again.

3

View your stats

All posts, impressions, and AI feedback appear in your dashboard. Data lives in ~/.post-pulse/ — never uploaded anywhere.
